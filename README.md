## ✨ Rule Categories
- **Completeness**: Null checks, required fields, minimum coverage thresholds
- **Uniqueness**: Primary key validation, deduplication rates
- **Validity**: Type checks, range validation, regex patterns, referential integrity
- **Timeliness**: Freshness SLAs, expected update frequency
- **Consistency**: Cross-table reconciliation, aggregation checks
- **ML Anomaly**: Statistical process control, isolation forest for outlier detection
